# **Passo a passo detalhado (versão escrita):**

1. O Windows Server vem com um período de testes gratuito. Nos nossos templates, a licença vem com apenas 3 dias, mas você pode prorrogar este tempo para até 365 dias gratuitos ativando o período de testes do Windows.

2. Depois de baixar o aplicative abra e digite o nome ou escolha o jogo ou serviço que você irá hospedar na sua VPS.

3. Aperte enter e Reinicie o seu servidor para aplicar corretamente as configurações.
