# **Passo a passo detalhado (versão escrita):**

1. O Windows Server vem com um período de testes gratuito. Nos nossos templates, a licença vem com apenas 3 dias, mas você pode prorrogar este tempo para até 180 dias gratuitos ativando o período de testes do Windows.

2. Para isto, basta abrir o "CMD" do Windows Server, e digitar o seguinte comando:
```bash {style={backgroundColor:"rgba(255,0,0,0.1)"}}
slmgr.vbs -rearm
```
3. Ao digitar este comando o Windows exibirá uma mensagem de sucesso, e após reiniciado o servidor, a licença do Windows estará prorrogada por mais 180 dias gratuitos.

4. Aperte enter e Reinicie o seu servidor para aplicar corretamente as configurações.
